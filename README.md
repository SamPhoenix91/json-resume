# JSON Resume

My CV in JSON format using <https://jsonresume.org/>

## Requirements

- Node.js
- npm

## Commands

### Install dependencies

```bash
npm install
```

### Validate JSON

```bash
npm run test
```

### Build HTML

```bash
npm run build -- --theme jsonresume-theme-elequent
# or
npm run build -- --theme jsonresume-theme-paper
# or
npm run build -- --theme jsonresume-theme-even
# ...
```
